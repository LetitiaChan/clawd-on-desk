📋 配置步骤

  1. Windows 用户必做: 在 CodeBuddy 插件 → Hooks 管理 → 高级设置中，设置命令执行器路径为: C:\Program Files\Git\bin\bash.exe
  2. 逐个创建钩子: 为下面每个事件创建钩子，复制对应的 shell 脚本
  3. 验证配置: 完成后重启 VSCode 测试钩子是否生效
